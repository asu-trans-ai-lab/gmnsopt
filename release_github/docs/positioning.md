# Positioning

gmnsopt is an open, GMNS-native optimization package and **open-science benchmark ecosystem** for
transportation planning, operations, and research. It converts standardized GMNS networks, demand files,
operational controls, and scenario data into mathematical-programming and network-optimization models.

It starts from foundational models — shortest path, min-cost flow, traffic assignment, system-optimal
assignment — then extends to dynamic, stochastic, multimodal, and real-world constrained cases: signal
control, OD calibration (ODME), congestion pricing, network design, work-zone scheduling, resilience,
evacuation, freight routing, EV charging, and UAM–ground coordination.

Each case is a reproducible benchmark with input data, mathematical formulation, solver implementation,
optimization output, simulation validation, and GUI4GMNS visualization — so **students** learn optimization
foundations, **researchers** compare algorithms (exact / heuristic / neural / RL), and **agencies** see how
optimization decisions affect traffic, queues, accessibility, reliability, emissions, and equity.

**Differentiator:** classic benchmark + GMNS + dynamics + stochasticity + real-world constraints +
visualization. Unlike generic solver benchmarks (e.g. MIPLIB), transportation semantics are preserved rather
than flattened into anonymous MIPs.
