# Tutorial ladder (foundation → real world)

1. What is a decision variable on a GMNS network?
2. Flow conservation on node–link topology
3. Capacity and link performance (BPR/QVDF)
4. Path flow and traffic assignment (UE / system-optimal)
5. Binary decisions and Big-M for traffic operations (signals, lane use, work zones)
6. Dynamic queues and time-expanded networks
7. Scenario-based stochastic optimization (robust, CVaR)
8. Real-world constraints and infeasibility diagnosis (IIS)
9. Visualization and public explanation (GUI4GMNS)

Each tutorial maps a chapter of classical mathematical programming onto a GMNS-native case; see
`../dev/book_to_gmns_mapping.md`.
